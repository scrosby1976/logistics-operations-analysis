# Logistics Operations Analysis

End-to-end data analysis project using SQL and Python to evaluate driver performance, trip efficiency, and operational metrics.

## Project Overview

This project explores operational data from a logistics company to identify performance trends, efficiency opportunities, and key business metrics. Analysis is performed using Python (pandas) and SQL (DuckDB) within Jupyter Notebook.

## Data Source

Dataset: Logistics Operations Database (Kaggle)

## Tools Used

- Python (pandas)
- SQL (DuckDB)
- Jupyter Notebook

## How to Run

1. Install dependencies:
   pip install kagglehub pandas duckdb

2. Open the notebook in the `notebooks/` folder

3. Run all cells:
   The dataset will download automatically using kagglehub

## Initial Focus

- Driver performance (trip volume)
- Trip efficiency analysis
